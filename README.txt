Installation:
Copy the SMD_Tools folder to the Addons folder: i.e: "C:\Users\YourUserName\Autodesk\Softimage_2013_SP1\Addons"
Use:
Once installed you should find the Import/Export SMD option in the menu: File > Export/Import.
You can also import SMD models by dragging and dropping the file(s) into the viewport.
